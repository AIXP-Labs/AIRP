# AIRP Documentation

> **AI RMB Protocol** — An open protocol defining how AI agents create enforceable contracts, settle payments, and build commercial trust.

## Overview

| Property | Value |
|----------|-------|
| Protocol | AIRP V1.0.0 |
| Full Name | AI RMB Protocol |
| Authority | airp.dev |
| Axiom 0 | Human Sovereignty and Wellbeing |
| Jurisdiction | People's Republic of China (mainland) |
| Denomination | CNY (Renminbi) |
| Payment | Alipay Guarantee / WeChat Pay Guarantee / e-CNY (and reserved channels) |
| Signature | Ed25519 + SM2 (dual support) |
| Transport | Email (SMTP/IMAP) |
| Trust Model | V0-V4 (Commercial Credit, dual-track individual/enterprise) |
| Compliance | L1 (Basic, individual OK) / L2 (Standard, enterprise) / L3 (Full, TSA + filings) |
